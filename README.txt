HƯỚNG DẪN SỬ DỤNG WEB ÔN THI 5 BỘ ĐỀ

1. Giải nén file zip.
2. Mở file index.html bằng Chrome, Edge hoặc Cốc Cốc.
3. Chọn một trong 5 bộ đề để bắt đầu làm bài.
4. Mỗi đề có 50 câu và thời gian 60 phút.
5. Sau khi chọn đáp án, câu hỏi sẽ bị khóa:
   - Chọn đúng: đáp án hiện màu xanh.
   - Chọn sai: đáp án bạn chọn hiện màu đỏ, đáp án đúng hiện màu xanh.
6. Bấm “Nộp bài” để xem số câu đúng/sai/chưa làm.
7. Đáp án A/B/C/D được đảo ngẫu nhiên mỗi lần bấm “Làm lại từ đầu”.

Các file chính:
- index.html: trang web chính.
- style.css: giao diện.
- app.js: logic làm bài, chấm điểm, đồng hồ.
- data.js: dữ liệu 5 bộ đề.

Lưu ý: Web chạy offline, không cần mạng internet. Tiến trình làm bài được lưu trên trình duyệt bằng localStorage.
